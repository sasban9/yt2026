The power session

Once a week:
the 3-hour batch day.

Hour
What you do
Output
Hour 1
Pull the week's scripts + drafts from your folder. Edit, trim, finalize 7 pieces.
7 finished scripts or post copies
Hour 2
Match visuals to each piece. Generate any missing images. Add captions & subtitles.
7 fully assembled posts
Hour 3
Load all 7 into Buffer/Later. Set publish dates, times, platforms. Write hashtags.
7 days of content scheduled

One session. Seven days handled. You never miss a day because you're never creating the day before.
